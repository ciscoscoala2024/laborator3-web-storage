# Laborator 3 - Web Storage & Cookies

## Descriere
Proiect pentru gestionarea datelor pe client-side folosind cookies, localStorage și sessionStorage.

## Fișiere
- `index.html` - Pagina principală
- `preferences.html` - Setări utilizator
- `cookies-info.html` - Informații tehnice
- `login.html` / `register.html` - Autentificare
- `dashboard.html` - Panou utilizator
- `cart.html` - Coș cumpărături
- `style.css` - Stiluri și teme
- `js/cookies.js` - Funcții cookies
- `js/storage.js` - Funcții storage
